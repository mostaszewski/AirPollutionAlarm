# AirPollutionAlarm
The script is designed for monitoring the level of air pollution.

Locates users on the basis of IP address, then gets data from the nearest monitoring station and sends a report to the provided e-mail address.

## API Token
You need an API Token to make this script work.

You can obtain it for free, here:
http://aqicn.org/data-platform/token/

## Basic usage
1. Open 'config.ini' and save Your details there.
2. Run a script.
